# Glenaeon Concordia Kit

Professional website implementation with mega menu navigation.

## Tech Stack
- HTML5
- CSS3 (LESS preprocessor)
- JavaScript (Vanilla)
- Bootstrap 5.3.2

## Setup

1. Install dependencies:
```bash
npm install
```

2. Compile LESS:
```bash
npm run build
```

3. Watch LESS files:
```bash
npm run watch
```

4. Run dev server:
```bash
npm run dev
```
